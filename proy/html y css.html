HTML y CSS

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para la creación de páginas web. Se utiliza para estructurar el contenido de un sitio web a través de etiquetas que definen elementos como textos, imágenes, enlaces y más.

Elementos básicos de HTML:

  • Estructura básica: Cada documento HTML comienza con la etiqueta <html> y contiene un <head> y un <body>.
  • Encabezados: Etiquetas <h1> a <h6> para títulos.
  • Párrafos: Se utilizan las etiquetas <p> para añadir texto.
  • Imágenes: La etiqueta <img> se usa para incluir imágenes.
  • Enlaces: Los enlaces se crean con la etiqueta <a>.
  • Listas:
    • Listas ordenadas: <ol>.
    • Listas desordenadas: <ul>.

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje de diseño utilizado para controlar la apariencia de los elementos en una página web. Permite agregar colores, fuentes, espacios, tamaños y animaciones.

Elementos básicos de CSS:

  • Selección de elementos:
    • Elementos: Selección directa por etiquetas, como p.
    • Clases: Selección usando un punto, como .clase.
    • ID: Selección con un numeral, como #id.
  • Propiedades comunes:
    • color: Cambia el color del texto.
    • background-color: Establece el color de fondo.
    • font-size: Ajusta el tamaño de la fuente.
    • margin y padding: Controlan el espaciado interno y externo de los elementos.
  • Animaciones: Con @keyframes se pueden crear efectos visuales dinámicos.
