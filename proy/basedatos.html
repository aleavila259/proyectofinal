Bases de Datos Relacionales

Tipos de Bases de Datos

Existen diferentes tipos de bases de datos según su estructura y propósito. Algunos de los más comunes son:

  • Bases de datos relacionales: Organizan los datos en tablas relacionadas, como MySQL, PostgreSQL y SQLite.
  • Bases de datos no relacionales: También conocidas como NoSQL, almacenan datos en formatos como documentos, grafos o claves-valor. Ejemplos: MongoDB, Cassandra.
  • Bases de datos en memoria: Diseñadas para operaciones rápidas almacenando los datos en la memoria RAM. Ejemplo: Redis.
  • Bases de datos en la nube: Servicios que ofrecen almacenamiento y gestión de datos en plataformas en la nube. Ejemplo: Firebase, Amazon RDS.

Ventajas de Usar Bases de Datos

  • Organización eficiente de grandes volúmenes de información.
  • Facilidad para realizar consultas y obtener datos específicos.
  • Seguridad en el almacenamiento de datos sensibles.
  • Posibilidad de realizar copias de seguridad y restauraciones.
  • Compatibilidad con múltiples lenguajes de programación.

SQL: Lenguaje para Bases de Datos Relacionales

SQL (Structured Query Language) es el lenguaje estándar para interactuar con bases de datos relacionales. Permite realizar operaciones como:

  • SELECT: Recuperar datos.
  • INSERT: Agregar nuevos registros.
  • UPDATE: Modificar datos existentes.
  • DELETE: Eliminar registros.

        -- Ejemplo básico de consulta SQL
        SELECT nombre, correo FROM usuarios WHERE edad > 18;

Configuración de Base de Datos en MySQL

Para configurar una base de datos en MySQL, sigue estos pasos:

  1. Accede a phpMyAdmin o usa el cliente MySQL desde la terminal.
  2. Crea una base de datos: 
    CREATE DATABASE educativa;
  3. Crea una tabla para almacenar datos: 
    
        CREATE TABLE usuarios (
            id INT AUTO_INCREMENT PRIMARY KEY,
            nombre VARCHAR(100) NOT NULL,
            correo VARCHAR(100) UNIQUE NOT NULL,
            fecha_registro TIMESTAMP DEFAULT CURRENT_TIMESTAMP
        );
